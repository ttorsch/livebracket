**Icon** — Lucide-derived line icons (2px rounded stroke). Substituted for the source product's Apple SF Symbols, which aren't redistributable.

```jsx
<Icon name="search" size={20} />
<Icon name="location" color="var(--color-primary)" />
<Icon name="starFilled" />
```

Inherits `currentColor`. Available: search, mic, location, star, starFilled, arrowRight, chevronRight, chevronDown, x, trophy, users, calendar, bell, plus, filter, play. Default size 20, strokeWidth 1.75. For icons beyond this set, pull the matching glyph from Lucide (lucide.dev) to stay consistent.
